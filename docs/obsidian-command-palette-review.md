# Obsidian 명령어 팔레트 전수 검토 (2026-07-07)

Obsidian 명령어 팔레트 화면 기록(총 97개 항목)을 전수 판독하고, Drive Obsidian Editor 적용 여부를 판정한 문서.

판정 기준:

- **적용** — 우리 제품에 도입한다. "이미 있음" 비고가 붙은 항목은 기존 기능이 충족하므로 추가 작업 없음.
- **변형** — 개념을 우리 제품에 맞게 바꿔 도입한다 (예: 시스템 경로 → vault 경로).
- **후순위** — 가능하지만 지금 하지 않는다. 사유 명시.
- **제외** — 제품 범위 밖(README 비포함 명시)이거나 브라우저 환경에서 불가.

구현 계획: [superpowers/plans/2026-07-07-slash-commands-shortcuts.md](superpowers/plans/2026-07-07-slash-commands-shortcuts.md) — 적용(이미 있음 제외)과 변형 전체를 다룬다.

브라우저 제약: ⌘1~9, ⌘W, ⌘N, ⌘T, ⌘,는 Chrome 예약 단축키라 가로챌 수 없다. 대체 키는 ⌥ 조합을 쓴다.
