# v0.2.0 릴리스 노트 (2026-07-06 ~ 07-07)

Drive Obsidian Editor가 "파일이 열리는 편집기"에서 "Obsidian처럼 쓰는 편집기"로 크게 확장되었습니다.

## 라이브 프리뷰 (신규)

문법 기호가 사라지고 서식이 바로 보이는 편집 모드가 기본이 되었습니다. 커서를 올리면 원문 마크다운이 다시 나타나고, 파일은 항상 원문 그대로 저장됩니다 — Obsidian과 완전히 호환됩니다.

- 제목·볼드·이탤릭·취소선·인라인 코드·인용구·수평선 렌더링
- ==하이라이트==, `> [!note]` 콜아웃(타입별 색상), 표(실제 테이블), 외부 이미지 인라인 표시
- 코드블록 언어별 구문 하이라이팅 (필요한 언어만 지연 로드)
- 체크박스 클릭으로 완료 토글
- 우상단 `</>` 버튼 또는 ⌘E로 원본 텍스트 모드 전환

## 링크로 이동하는 vault

- **위키링크 클릭 이동** — `[[노트]]`를 클릭하면 해당 노트가 열립니다. 외부 링크는 새 탭.
- **없는 노트는 점선으로 표시**되고, 클릭하면 그 자리에서 새 노트가 생성됩니다.
- **⌘[ / ⌘]** 로 문서 이동 히스토리를 오갑니다.
- **⌘O 빠른 전환기** — 최근 파일 우선으로 파일명 점프.

## 작성 도구

- 슬래시 커맨드 15종 (제목/목록/체크박스/인용구/코드 블록/표/콜아웃 등) — 한글 이름으로 검색됩니다.
- 편집 단축키: ⌘B 볼드, ⌘I 이탤릭, ⌘K 링크, ⌘L 목록↔체크박스 순환, ⌘⇧1~6 제목 설정/해제, ⌘Z 실행 취소, ⌥↑/↓ 줄 이동, Tab 들여쓰기, ⌘F 문서 내 찾기/바꾸기
- Enter 시 목록·체크박스 자동 이어가기, `#` 입력 시 태그 자동완성
- **⌥T 템플릿 삽입** — `templates/` 폴더의 노트를 `{{title}}` `{{date}}` `{{time}}` 치환해 삽입
- **⌥D 일일 노트** — 오늘 날짜 노트 열기/생성 (daily 템플릿 지원)
- 헤딩 접기, 읽기 좋은 줄 길이(중앙 정렬)
- 사이드바 하단 **? 버튼**에서 전체 단축키 목록 확인

## Drive라서 가능한 기능

- **버전 기록** — 에디터 헤더 시계 아이콘에서 Drive 버전 열람·미리보기·복원
- **본문 전문 검색** — 사이드바 검색이 파일명뿐 아니라 노트 내용까지 찾습니다
- **원격 변경 감시** — 열려 있는 문서가 다른 기기에서 수정되면 30초 안에 경고
- **파일 이동** — 사이드바 ··· 메뉴에서 다른 폴더로 이동
- 경로 복사, Drive 웹에서 바로 열기

## 안정성·성능

- 토큰 만료 시 자동 재발급, Drive 요청 실패(429/5xx) 지수 백오프 재시도 — 장시간 세션에서 저장 실패 감소
- 마지막으로 열었던 문서를 다음 실행 때 자동 복원
- 문서 내용 캐시 + 사이드바 호버 프리페치로 파일 열기 지연 감소
- 언어 파서 지연 로드로 초기 번들 절반 축소 (1.6MB → 0.36MB)
- 모든 다이얼로그가 배경 클릭·Esc로 닫힘

## 앱 단축키 모음

| 키 | 동작 |
|---|---|
| ⌘S | 저장 |
| ⌘E | 라이브/원본 모드 전환 |
| ⌘O | 빠른 전환기 |
| ⌘⇧F | 파일 검색 |
| ⌘[ / ⌘] | 이전/다음 문서 |
| ⌥N / ⌥T / ⌥D | 새 노트 / 템플릿 / 일일 노트 |
| ⌥1~9 | 최근 탭 이동 (9=마지막) |

(Windows/Linux는 ⌘ 대신 Ctrl)
